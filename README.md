# Data Visualization Excercises
This repository is for EDS 240 Data Visualization and Communication. This course focuses on using data visualization principles and programmatic tools to communicate data science work responsibly, clearly, and effectively to diverse audiences. This repository houses my various projects as I develop my data visualization skills. 

## Contributor
